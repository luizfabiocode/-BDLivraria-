# 📚 Projeto BDLivraria

## 🎯 Objetivo
Este projeto acadêmico tem como finalidade a criação de um banco de dados para uma livraria, explorando conceitos de **modelagem**, **normalização** e **consultas SQL**.

## 🛠️ Tecnologias utilizadas
- SQL
- MySQL

## 📂 Estrutura do projeto
- `scripts/` → Arquivos SQL para criação de tabelas
- `inserts/` → Scripts de inserção de dados
- `queries/` → Exemplos de consultas
